Sophie's Random Color Palette Inspiration Tool

Click the button below to generate a new color palette.

Once a palette is generated, you can change individual colors by clicking on them. You can either change a color randomly by pressing the Random Color button, or manually by using the color picker. These options show up when you hover or click on the color.

Want to Print Your Palette?

If you do not see any colors in the Print Preview after pressing "Print Palette", make sure to check the "Background Graphics" option (if you're using Google Chrome). Otherwise none of the palette colors will show up, only the hex codes! In Google Chrome, this option is under "More Settings".
